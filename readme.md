# Speedwing
A new version of speedwing written in rust!

I'm new to rust so i don't really know what i'm doing, my code probably is knowhere optimal, but i'm sure it'll get better as time goes on.

The game is deployed on https://speedwing.silverspace.io

### Controls:
- Move mouse to move player
- Left click to lock mouse
- ESC to unlock mouse
- \ to go back to menu
- ; to toggle debug mode
- ' to toggle editor mode

#### Editor
- WASD to move camera
- Left button to create
- Right button to destroy
